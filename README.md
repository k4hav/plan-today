## ✨ Features

- 🎯 **Goal-Based Categories** — JEE, NEET, UPSC, GATE, CAT, Work, Fitness, SSC, Custom
- 📋 **Task Board** — Add, complete, miss, delete tasks with animated checkboxes
- 📓 **Notes & Reflections** — Save daily learnings, doubts, achievements
- 📊 **Analytics** — 30-day heatmap, weekly bar chart, category distribution
- 🍅 **Pomodoro Timer** — 25/5 min focus sessions with animated ring
- 📅 **Week Calendar** — Visual 7-day progress tracker
- 🔥 **Streak Tracker** — Consistency streak counter
- 🗂 **Custom Categories** — Add your own subjects/topics
- 💾 **Persistent Storage** — Data saved in localStorage (never lost)
- ⚡ **Animated Logo** — Custom SVG calendar with drawing animations
- 🎨 **3D Card Effects** — CSS perspective transforms on hover
- 💫 **Ripple Buttons** — Material-style click animations
- 🌙 **Dark Theme** — Easy on the eyes, always

## 📁 Project Structure
```
plantoday/
├── index.html   → HTML structure & layout
├── style.css    → All styles, animations, variables
├── data.js      → Static data (goals, templates, quotes)
└── app.js       → All application logic & state
```

## 🚀 Getting Started

### Option 1 — Just open it
Download the files and open `index.html` in any browser. Done.

### Option 2 — Clone it
```bash
git clone https://github.com/k4hav/plantoday.git
cd plantoday
# Open index.html in browser
```

No build step. No npm install. No terminal needed.
